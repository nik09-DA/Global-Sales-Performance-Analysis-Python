# Global-Sales-Performance-Analysis-Python
"End-to-end sales analysis using Python for data cleaning and Power BI for interactive dashboarding."
### Project Overview###
This project focuses on analyzing global sales data. I used Python for the heavy data cleaning and Power BI for the final interactive dashboard.

## What I did in Python:
Libraries Used: Pandas (for cleaning) and Matplotlib/Seaborn (for data checks).

Data Cleaning: * Handled Null Values in 'State' and 'Territory' columns.

Converted 'Order Date' into a proper Datetime format.

Removed duplicates and fixed data types to make it "Analysis-Ready."

## What I did in Power BI:
Visualization: Imported the cleaned data from Python and built a professional dashboard.

Features: * Used Drag-and-Drop visuals (Bar charts, Line graphs).

Added Slicers for Country and Product Line for easy filtering.

No complex DAX used as the data was already cleaned in Python.

Key Insights:
Top Product: Classic Cars are the main revenue generator.

Top Market: The USA leads in total sales volume.

Sales Trend: November is the peak sales month every year.
